# Role: Senior Scrollytelling UX Lead
# Project: Marianne Cottage - "The Tactile Journey"

# Concept
A vertical narrative journey. The user doesn't just "browse"; they "uncover" 
the cottage history through an immersive, scroll-triggered experience.

# Visual Requirements
- **Transition Logic:** Use Intersection Observer to trigger opacity shifts. 
  Start with a 1800s "Charcoal Sketch" of the house that colorizes into a 
  lush 2026 photograph as the user scrolls.
- **Color Transitions:** Shift from #3A3A3A (Historical B&W) into a 
  vibrant #A9B7AC (Modern Sage) as the story moves to the present day.
- **The "Liquid Glass" Menu:** A navigation bar that stays hidden until the 
  user reaches the "Modern Comforts" section, then slides in with a 
  blurred-glass effect.

# Task
1. Provide the CSS/JS for a "Scroll-to-Colorize" hero section.
2. Define the typography scale for "Big Storytelling" headers.
3. Build a "Booking reveal" footer that feels like an organic conclusion 
   to the cottage's history.
