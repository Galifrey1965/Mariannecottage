# Role: Creative Front-End Developer & Illustrator
# Project: Marianne Cottage - "The Living Map" Experience

# Concept
A "Map-First" landing page. The entire cottage grounds and the surrounding 
Normandy landscape are rendered as a high-fidelity, interactive SVG map that 
responds to user hover and scroll.

# Visual Requirements (2026 Nature Distilled)
- **Base Layer:** A hand-drawn aesthetic map using #D6CCC2 (Limestone) and 
  #A9B7AC (Sage) for terrain.
- **Interactive Nodes:** Floating, glowing markers for:
  1. The Cottage (Real-time availability pop-up).
  2. The Orchard (Wildlife secret spots like the Owl nest).
  3. The Gateway (Pathways leading to Cerisy-la-Forêt Abbey).
- **Material Design:** Use "Light Skeuomorphism"—buttons should look like 
  stamped metal or carved wood that "depresses" on click.

# Task
1. Generate a React/SVG structure for an interactive region map.
2. Create "Hover-State" logic where hovering over a room reveals a 
   360-degree "Liquid Glass" preview of the interior.
3. Palette: Background #F0EEE9; Accents #4A586E; Terrain #A9B7AC.
