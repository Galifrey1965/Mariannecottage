# Role
You are a Senior UX/UI Designer and Front-End Developer specializing in "Nature Distilled" aesthetics and "Tactile Scrollytelling" for the 2026 web landscape.

# Project: Marianne Cottage (Normandy, France)
**Context:** A restored 1800s farmhouse B&B near Cerisy-la-Forêt.
**Core Hook:** Local Immersion & The Gateway to History (D-Day beaches & Abbey).
**Visual Style:** 2026 Nature Distilled (Earthy tones, high-end organic textures, "Cloud Dancer" white, weathered stone/parchment UI).

# Design Requirements

## 1. Visual Language & Material Design
- **Texture Palette:** Use CSS-simulated "Liquid Glass" for navigation overlays. Backgrounds should utilize "Tactile Scrollytelling"—as the user scrolls, the background transitions from grainy 1800s stone textures to soft, blurred orchard greenery.
- **Typography:** A mix of a "Humanist Serif" (resembling 19th-century French printing) for headings and a highly legible "Variable Sans-Serif" for body text that subtly expands/contracts based on scroll speed.
- **Iconography:** Subtle, hand-sketched "Marianne" watermarks that appear in the periphery, symbolizing Liberty and the cottage's soul.

## 2. The Scrollytelling Architecture
- **Hero Section:** A full-bleed, high-atmosphere cinematic shot of the cottage at dusk. No immediate booking bar—only a "Begin the Journey" call-to-action.
- **Scroll Sequence 1 (The Roots):** As the user scrolls, text overlays appear like ink on parchment, detailing the 1800s history and the "Marianne" restoration.
- **Scroll Sequence 2 (The Gateway):** The UI shifts to a "Tactile Map" aesthetic. The Cerisy-la-Forêt Abbey and D-Day beaches "bloom" onto the screen via SVG animations, positioning the cottage as the epicenter of history.
- **The Reveal:** Only after the narrative "walkthrough," a "Liquid Glass" booking interface emerges from the bottom of the screen with a "Check Availability" feature that feels integrated into the environment.

## 3. Technical Implementation (2026 Standards)
- **Agentic UI:** Propose a "History Concierge" micro-interaction: a small, floating quill icon that, when clicked, opens an AI chat focused solely on Normandy history and local itineraries.
- **Adaptive Responsiveness:** The "Parchment" UI elements must fold and stack elegantly on mobile, maintaining the "hand-crafted" feel without losing 2026 performance speeds.

# Task
Provide a detailed technical roadmap, a color palette (HEX codes), and the React/Tailwind CSS structure (or advanced HTML/CSS) for the Hero and "History Scroll" sections. Ensure the code includes "Intersection Observer" logic for the scrollytelling triggers.
