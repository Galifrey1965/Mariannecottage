# Role: Expert Agentic UI Designer
# Project: Marianne Cottage - "The Historian Concierge" Interface

# Concept
The website is a high-end AI portal. The hero section is a minimalist, cinematic 
view of the cottage. The primary interaction is an AI "Historian" that guides 
the user through the B&B's heritage and the Normandy region.

# Visual Requirements (2026 Nature Distilled)
- **Background:** #F0EEE9 (Cloud Dancer) with a subtle "Paper Grain" CSS texture.
- **The Concierge UI:** Instead of a chat bubble, use a "Quill & Ink" floating 
  action button (FAB) that expands into a parchment-textured sidebar (#D6CCC2).
- **Typography:** Titles in 'Benton Modern' (Serif); Agent responses in a clean 
  modern Sans-Serif (#3A3A3A).
- **Micro-interactions:** When the AI mentions a location (e.g., "Omaha Beach"), 
  the main site background should subtly morph to a blurred, desaturated 
  photograph of that location.

# Task
1. Design the "Agentic Dashboard" where the AI suggests local itineraries.
2. Provide Tailwind CSS for a "Glassmorphic Parchment" container.
3. Include a "Tactile" booking button that feels like a physical wax seal.
